## Accessing the Admin Dashboard

### Login

1. Navigate to: `https://your-domain.com/admin/dashboard`
2. Enter your credentials:

## Understanding the Dashboard

### Dashboard Overview

The dashboard shows three categories of requests:

#### 1. Pending Requests (Yellow)
- New submissions from clients
- No designs uploaded yet
- Require immediate attention

#### 2. In Progress (Blue)
- Currently being worked on
- May have partial designs uploaded
- Waiting for completion

#### 3. Completed (Green)
- All designs uploaded
- Client has been notified
- May include client feedback

### Request Cards

Each request card shows:
- Company name and contact person
- Submission date
- Number of designs uploaded
- Quick action buttons

## Processing New Requests

### Step 1: Review New Request

1. In the "Pending Requests" section, click **"View Details"**
2. Review the following information:
   - **Client Information**: Contact details
   - **Brand Profile**: Keywords, colors, target audience
   - **Client Logo**: If uploaded, review for design inspiration
   - **Additional Notes**: Any special requests

### Step 2: Update Status

1. In the sidebar, find "Update Status"
2. Change from "Pending" to "In Progress"
3. Click **"Update Status"** button

This indicates you're actively working on this request.

### Step 3: Plan Your Designs

Based on the brand profile, plan 3-4 design concepts:

**Example Process:**
- Brand Keywords: "Innovation, Modern, Sustainable"
- Brand Colors: Blue and Green
- Target Audience: Tech companies

**Design Strategy:**
1. **Design 1**: Sleek modern tech aesthetic
2. **Design 2**: Nature-inspired sustainable theme
3. **Design 3**: Bold innovative look
4. **Design 4**: Balanced professional approach

## Creating and Uploading Designs

### Using AI Tools (Recommended Workflow)

#### Option 1: Midjourney

1. Use the brand keywords to create prompts:
   ```
   /imagine corporate gift design, [brand keywords],
   modern professional, [brand colors], high quality,
   product photography style
   ```

2. Generate 4+ variations
3. Select the best 3-4 options
4. Upscale and download

#### Option 2: DALL-E

1. Create detailed prompts:
   ```
   A professional corporate gift design featuring [product],
   incorporating [brand keywords], using [brand colors],
   clean and modern style, high-end product photography
   ```

2. Generate multiple variations
3. Download selected options

#### Option 3: Canva Pro

1. Start with a template
2. Customize with brand colors
3. Incorporate brand keywords into design
4. Export as PNG or JPG

### Uploading to Platform

1. In the request detail page, scroll to "Upload Design Options"
2. Fill out the form:
   - **Design Title**: Brief, descriptive name
     - Example: "Modern Tech Aesthetic"
   - **Design Description**: Explain the concept (2-3 sentences)
     - Example: "This design emphasizes innovation with clean lines and a tech-forward approach. The blue color scheme reflects your brand's professional identity."
   - **Design Image**: Upload your file (PNG, JPG, or PDF)
3. Click **"Upload Design"**
4. Repeat for all design options (aim for 3-4 total)

### Design Guidelines

**Image Specifications:**
- **Format**: PNG or JPG (PNG preferred for transparency)
- **Resolution**: Minimum 1920x1080px
- **File Size**: Under 15MB
- **Aspect Ratio**: 16:9 or 1:1 works best
- **Quality**: High-resolution, professional-looking

**Title Guidelines:**
- Keep it short (3-5 words)
- Descriptive and appealing
- Examples:
  - "Classic Elegance"
  - "Bold & Innovative"
  - "Nature-Inspired Sustainable"
  - "Modern Minimalist"

**Description Guidelines:**
- 2-4 sentences
- Explain the design concept
- Connect to brand keywords
- Highlight unique features
- Example:
  ```
  This design captures your brand's commitment to sustainability
  through earth-tone colors and organic shapes. The clean layout
  appeals to environmentally-conscious clients while maintaining
  a professional appearance.
  ```

## Managing Design Options

### Editing Designs

After uploading, you can update design information:

1. Find the design in the "Uploaded Designs" section
2. Edit the title or description in the form fields
3. Click **"Update"**

### Deleting Designs

If you need to remove a design:

1. Click the **"Delete"** button on the design
2. Confirm the deletion
3. Upload a replacement if needed

**Note**: Deleted designs cannot be recovered.

### Organizing Designs

Designs are displayed in the order they were uploaded. Upload them in your preferred order of presentation.

## Generating PDF Presentations

### When to Generate PDF

Generate a PDF when:
- All 3-4 designs are uploaded
- Descriptions are complete and polished
- Ready to present to client

### Generating the PDF

1. In the request detail page, click **"Generate PDF"**
2. The PDF will download automatically
3. Review the PDF before sending to client

### PDF Contents

The generated PDF includes:

**Page 1: Cover**
- TezzaWorks branding
- Client company name
- Date

**Page 2: Brand Profile**
- Client's brand keywords
- Brand colors
- Target audience
- Any additional notes

**Pages 3+: Design Options**
- Each design on its own page
- Title and description
- High-quality image

**Final Page: Next Steps**
- Review process explanation
- Contact information
- Next steps outline

### PDF Best Practices

- Review the PDF before sending
- Ensure all images loaded correctly
- Check that descriptions are complete
- Verify client information is accurate

## Communicating with Clients

### Initial Notification

When designs are ready:

1. In the request detail page, find "Gallery Link" in the sidebar
2. Click **"Copy Link"**
3. Draft an email to the client:

```
Subject: Your Custom Design Options Are Ready - TezzaWorks

Hi [Contact Name],

Great news! Your custom design options are ready to view.

We've created [X] unique design concepts based on your brand
profile, each reflecting [mention key keywords].

View Your Design Gallery:
[Paste Gallery Link]

You can review the designs, select your favorites, and provide
feedback directly through the gallery.

I've also attached a PDF presentation for your convenience.

Looking forward to your thoughts!

Best regards,
[Your Name]
TezzaWorks Design Team
info@tezzaworks.com
(555) 123-4567
```

4. Attach the generated PDF
5. Send the email

### Responding to Feedback

When client submits feedback:

1. Review their selected designs (marked with hearts)
2. Read their comments
3. Note their star rating
4. Respond based on feedback:

**If satisfied (4-5 stars):**
- Thank them for feedback
- Discuss next steps (refinement, production)
- Provide pricing information

**If requesting changes (1-3 stars):**
- Acknowledge their feedback
- Ask clarifying questions
- Offer to create revised options
- Update status back to "In Progress" if needed

## Best Practices

### Quality Standards

1. **Always upload 3-4 options** (not more, not less)
   - Gives clients choice without overwhelming them
   - Shows effort and consideration

2. **Maintain consistency** across designs
   - All should reflect the brand profile
   - Professional quality throughout

3. **Write compelling descriptions**
   - Connect design to brand values
   - Explain the concept clearly
   - Keep it concise but informative

### Time Management

1. **Respond to new requests within 24 hours**
   - Update status to "In Progress"
   - Send acknowledgment if needed

2. **Complete designs within 2-3 business days**
   - Standard turnaround time
   - Communicate if delays occur

3. **Follow up on feedback within 24 hours**
   - Shows responsiveness
   - Keeps momentum going

### Client Communication

1. **Be professional and friendly**
2. **Set clear expectations**
3. **Provide context for designs**
4. **Ask clarifying questions** when needed
5. **Document all communications**

### Design Strategy Tips

1. **Variety is key**: Offer different approaches
2. **Stay on brand**: Don't stray too far from brand profile
3. **Consider practicality**: Think about production
4. **Show range**: Different styles within the brand aesthetic

### Workflow Efficiency

1. **Batch similar tasks**: Process multiple requests' designs together
2. **Create templates**: Save time on common design elements
3. **Use keyboard shortcuts**: Learn the platform
4. **Keep organized**: Note which clients need follow-up

## Common Scenarios

### Scenario 1: Client Requests Changes

**Situation**: Client loves the concept but wants different colors.

**Action**:
1. Update status back to "In Progress"
2. Create revised versions
3. Upload as new designs (or replace old ones)
4. Notify client of updates

### Scenario 2: Client Can't Decide

**Situation**: Client loves multiple options equally.

**Action**:
1. Suggest creating variations combining elements
2. Offer to create mockups in different contexts
3. Provide guidance on selection criteria
4. Schedule a call if needed

### Scenario 3: Client Needs Rush Service

**Situation**: Client needs designs urgently.

**Action**:
1. Assess feasibility
2. Communicate realistic timeline
3. Consider simpler design approach for speed
4. Prioritize this request

### Scenario 4: Unclear Brand Direction

**Situation**: Brand keywords are vague or contradictory.

**Action**:
1. Email client for clarification
2. Ask specific questions:
   - What companies do you admire?
   - What emotions should the design evoke?
   - Any examples you love/hate?
3. Wait for response before proceeding

## Troubleshooting

### Upload Fails

**Solution:**
- Check file size (under 15MB)
- Verify file format (PNG, JPG, PDF only)
- Try refreshing the page
- Contact tech support if persists

### PDF Generation Error

**Solution:**
- Ensure all designs are uploaded
- Check that images are valid
- Verify descriptions are filled in
- Try generating again

### Can't Access Request

**Solution:**
- Refresh the page
- Check your login status
- Verify the request exists
- Contact system admin
